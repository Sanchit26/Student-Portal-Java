# рџљЂ Getting Started

## Prerequisites

- Java JDK 21+ (or newer)
- Apache Maven
- Git

---

## Installation

Clone the repository

```bash
git clone https://github.com/<your-username>/Student-Portal-Java.git
```

Move into the project directory

```bash
cd Student-Portal-Java
```

Compile the project

```bash
mvn clean compile
```

Run the application

```bash
mvn exec:java
```

Open your browser

```
http://localhost:8080
```
